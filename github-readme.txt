# Tarjeta de San Valentín

Una hermosa tarjeta de San Valentín interactiva con animaciones y efectos visuales.

## Características
- Animación de corazones flotantes
- Mensajes aleatorios de amor
- Efectos de transición suaves
- Diseño responsive

## Visualizar
Puedes ver la tarjeta en vivo aquí: https://[tu-usuario].github.io/valentine-github/

## Uso
1. Clona este repositorio
2. Personaliza los mensajes en `assets/js/script.js`
3. Modifica el nombre en el archivo `index.html`
4. Despliega en GitHub Pages
